# Setup

This sample project is intentionally minimal. To exercise the quickstart in
your own checkout:

1. Build both binaries (`go build` + `cargo build`).
2. Run `contextforge init`.
3. Import the Hermes memory fixture next door.
4. Index this directory.
5. Search.

The `configuration` keyword in `docs/config.md` doubles as a search target so
the quickstart's `search` step returns a deterministic hit.
