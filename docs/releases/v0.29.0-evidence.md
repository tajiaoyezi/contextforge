### 3. exact-below-threshold 诚实 caveat（ADR-013，MUST 记录）

**recall@10=1.0000 是活体 KNN 的「正确性证明」，不是 HNSW 逼近态的压测**：在 N=2000（**低于** qdrant HNSW `indexing_threshold` 默认 ~10000）时，qdrant 直接服务 **EXACT KNN**（未构建 HNSW 近似索引）——所以此处实测的是「活体 qdrant KNN **== brute-force 精确 ground truth**」这一正确性契约（活体 == 精确），替换原 `eval_integration.rs` 0.7/0.85 合成 fixture。recall=1.0 反映的是「活体后端在小语料下与精确 KNN 逐位一致」，**不**代表 HNSW 近似算法被压测过。压测 **HNSW 逼近态**（大语料 > indexing_threshold + optimizer 实际建出近似索引）据真实诚实延后 `[SPEC-DEFER:phase-future.vector-large-corpus-perf]`。**不声称 HNSW approximation 被压测**。
