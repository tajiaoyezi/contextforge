# ContextForge v0.12.0 Artifacts — vector-retrieval-integration (end-to-end semantic search + ADR-023 ratified)

> Artifact inventory for the v0.12.0 minor release (Phase 19). Mirrors the v0.11.0 artifacts
> structure. The default build is unchanged and dependency-free; the real embedding model is a
> feature/deploy choice (`embedding-fastembed`).

## 1. 二进制档

- `contextforge`（Go 控制面 CLI）：含 `eval run --semantic` 双路 + `console-api-serve` REST proxy。
- `contextforge-core`（Rust 数据面 daemon）：默认构建 = BM25 + 0-dep `BruteForceVectorBackend` 语义路径 + `DeterministicEmbeddingProvider`（0 新 dep）。
- Feature 构建：`--features embedding-fastembed`（real `all-MiniLM-L6-v2` provider，rustls + ONNX）；`--features vector-hnsw`（D2 跨平台 ANN backend）等 `vector-*` feature 见 ADR-023 D1-D5。
