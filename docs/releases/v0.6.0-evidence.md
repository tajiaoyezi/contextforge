## Upgrade path (v0.5.0 → v0.6.0)

1. `git fetch --tags && git checkout v0.6.0`
2. `cargo build -p contextforge-core --release && go build ./cmd/contextforge`
3. 重启 daemon：旧 daemon 进程 SIGTERM；新 binary 启动会在既有 data_dir 内
   create memory.db + apply 0013_memory_items.sql migration（IF NOT EXISTS 幂等）
4. Console UI 端：Memory panel v1.0 端到端调用代码已 ship；v0.6 ship 后 Console
   UI 可消费（fresh install 看 0 条 → graceful degrade 提示 "import via CLI"；
   importer 改造留 v0.6.x [SPEC-DEFER:phase-15.import-to-memory-items]）
5. smoke 验证：`bash scripts/console_smoke.sh` 期望 `CONSOLE_REAL_SMOKE_EXIT=0`
6. ops curl 工作流：调 deprecate / soft-delete 需加 `-H 'X-Confirm: yes'` 或 `?confirm=true`
