## 6. 版本声明（contract bump 评估）

- qdrant 生命周期层 + lancedb 索引调参参数均为 core 库 retriever/vector 层 feature-gated inherent 类型/方法（不改 task-18.1 三 trait `VectorBackend`/`VectorIndexer`/`VectorSearcher` 签名）。proto 零 delta；Console Contract v1 shape 不变；smoke `[34/34]` step add-only（既有 step 不破坏）。
- 默认构建 0 vector 依赖 + BM25-only baseline 不变（ADR-023 D5）。
- 故 **无 breaking contract bump**；既有客户端无需改动。生产 backend 经 feature opt-in（qdrant 需 live server / lancedb 需 protoc 前置）。
