## 6. 版本声明（contract bump 评估）

- `MemoryItem.pinned_by`/`pinned_at_unix`（field 11/12）+ `MemoryService.Unpin`/`HardDelete` RPC 全 add-only（既有 field 1-10 + 5 RPC + `Pin{bool pin}` 不动，proto-freeze guard 过）。`hard_delete`/`set_pinned_with_actor`/`reconcile_is_pinned_from_audit` 是 store add-only 方法（`set_pinned` 委托保签名）。SQLite migration add-only column with default。
- 默认构建 0 新依赖 / 0 network（ADR-004）。
- 故 **无 breaking contract bump**；既有 v0.6–v0.19 client 无需改动。unpin/hard-delete/backfill 均 add-only / opt-in。
