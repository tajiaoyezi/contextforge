## Trade-offs ship 矩阵 (v0.5.0)

| Decision | Status @ v0.5 | [SPEC-DEFER] target |
|---|---|---|
| Trace SQLite persistence | in-memory LRU cap=1000 (daemon restart 即丢) | `task-future.search-trace-sqlite-persistence` 留 v0.5.x |
| Chunk byte offsets | `chunk_offset_start/end = 0` (line ranges sufficient for Console UI) | `chunk-byte-offsets` 留 v0.5.x SchemaMigration |
| Multi-workspace strict isolation | 全局 chunk_id 唯一假设 | `phase-15.multi-workspace-strict` 留 v1.x |
| Cancel REST status code | 204 No Content (was 200) | hard locked by ADR-017 D3 |
| X-Confirm semantics | OR (header OR query) | hard locked by ADR-017 D2 |
| Long-poll vs SSE for events | long-poll only (v0.4 retained) | `task-future.consoleapi-sse` 留 v1.x |
| RFC3339Nano timestamps | kept (Console Zod relax follow-up) | hard locked by ADR-017 D5 |
| ADR-017 Status | **Proposed** | full **Accepted** at Phase 14 closeout (v0.7.0) |
