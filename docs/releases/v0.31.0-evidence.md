### 3. 3 次 run 全稳定 + CI honest-defer 诚实 caveat（ADR-013，MUST 记录）

**(a) MRR=1.0000 / recall@1=1.0000 是「明显相关项排在近义干扰之上」的证明，不是大基准质量断言**：作者手工标注集仅 **14 query×candidate case**（小型），MRR=1.0 / recall@1=1.0（3 次 run 全稳定）证明 real 远程 cross-encoder 把明显相关项稳定排到第 1、压过故意埋的硬近义干扰（config_save↔config_load、bm25↔hybrid、cjk_index↔cjk_vector、cosine↔vector_backend、cache↔chunk）——这是「远程 cross-encoder 重排在该标注集上正确」的端到端联调正确性证明，**不**代表大基准 / 标准基准 / 大规模标注集质量。大语料重排质量据真实诚实延后 `[SPEC-DEFER:phase-future.reranker-large-corpus-quality]`。

**(b) 3 次 run 全稳定（对照 Phase 37 embedding 的跨 run 波动）如实记录**：与 v0.30.0 Phase 37 embedding（recall@1 在 0.8667–0.9333 间**跨 run 波动**）**不同**——本 rerank 在 3 次 run 间 **MRR=1.0000 / recall@1=1.0000 完全稳定**：cross-encoder 对 query-candidate 的联合打分在该小标注集上**更具决定性**（相关项 relevance_score 与近义干扰拉开足够间距，3 次 run 排序一致），故无跨 run 波动。identity 基线 MRR=0.4762 / recall@1=0.0000（no-semantic 控制组：候选均匀分 → chunk_id tie-break，相关项从不字母序最先），delta_MRR=+0.5238 / delta_recall@1=+1.0000 即真实重排带来的增益。如实记录稳定性来源（cross-encoder 联合打分在小集上更决定性），**不**夸大为「任意语料皆稳定」。

**(c) CI honest-defer（关键诚实差异，与 qdrant 不同）**：remote reranker 是**付费外部 API、无免费 service container**——与 v0.29.0 的 qdrant **不同**（qdrant 有免费 OSS service container 在**每次 CI run** 守护活体召回）。本 harness 在 CI 无密钥时**干净 honest-defer SKIP**，真实重排**由本机已认证 run 实测**而**非每次 CI run 守护**。据 ADR-013 据实记录，**不**夸大为「CI 每次 run 守护重排」——真实重排数据来源是本机认证 run（SiliconFlow + Qwen3-VL-Reranker-8B，3 次 run），CI 仅护栏「无密钥时干净 SKIP 绝不 fail」+ harness 编译/结构门。复用既有 `[SPEC-DEFER:phase-future.embedding-remote-ci-credential]`（**不**新造 reranker-ci-credential——付费外部 API 无免费 container 的 CI 凭据缺口是同一条已记录的 defer）。
