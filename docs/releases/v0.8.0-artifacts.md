## 7. Trade-offs ship 矩阵 (Phase 15)

| Decision | Decision rationale | Cost / Future work |
|---|---|---|
| FIFO over LRU cache (task-15.1) | Single-process fallback demo充分 | LRU留 [SPEC-DEFER:phase-future.cache-lru] |
| memory.pin / memory.unpin 合并 event_type | event_type namespace 紧凑 | Future UI filter via payload.op |
| today_delta lexicographic compare | 复用既有 indexed_at TEXT 列 / 无 chrono dep | 高频写入时与 timezone-aware compare 一致 |
| TraceRecord wrapper (task-15.5) | RetrievalTrace proto 不动 (ADR-015 D1) | Crate-internal TraceStore.put 签名 BREAKING (no Go client impact) |
| embed config-only probe | 远程 ping rate-limit / secret-exposure 风险 | 远程 ping 留 [SPEC-DEFER:phase-future.embed-remote-probe] |
| synthesize fallback for nil HealthClient | Console UI 永远拿到完整 5 key shape | Production wiring 真 HealthClient via cli.Health() |
| 5 探针 sequential | 简单可靠，total < 100ms 实测 | Parallel 留 future optimization 如出现 retriever 慢的 case |
