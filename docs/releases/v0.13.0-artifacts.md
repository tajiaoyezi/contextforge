## 6. 版本声明（contract bump 评估）

- proto `console_data_plane SearchRequest.semantic`(7) 为 **add-only** 请求字段（task-20.1，`buf generate proto` 重生成 + proto-freeze 守护 PASS；Rust pb 经 `core/build.rs` tonic_build 自动重生成）。Console Contract v1 shape 不变；既有 22-endpoint conformance 不破坏；unset 时退回 BM25 行为。
- 故 **无 breaking contract bump**；既有客户端无需改动（仿 ADR-015/022 add-only pattern）。语义 hits 经既有 `score`（= 向量分）+ `retrieval_method="vector"` + trace `candidate_generation_steps=vector-bruteforce` 区分，未加独立 `vector_score` proto 字段（见 §7 follow-on）。
