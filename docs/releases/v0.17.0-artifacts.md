## 6. 版本声明（contract bump 评估）

- code/CJK tokenizer 是 core 库 indexer 层 opt-in 能力（`RetrieverConfig.tokenizer` 既有字段，task-4.1 留的接入点，本 release 首次据其名注册同名 analyzer）。proto 零 delta；Console Contract v1 shape 不变；smoke `[33/33]` step add-only（既有 step 不破坏）。
- eval `ValidateGoldenSemantic` add-only（既有 `ValidateDataset` 语义不动）。
- 故 **无 breaking contract bump**；既有客户端无需改动。opt-in tokenizer 不默认启用（采用须 re-index）。
